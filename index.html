Run Buddy

Run Buddy is a step-by-step tutorial on teaching the basic principles of HTML and CSS. Using provided mockups, the program teaches new web developers how to code and style new websites from scratch.

Lessons include:

  1. Basic HTML and CSS layouts
  2. Positioning and organization of elements
  3. Incorporation of Git and Github
  4. Flexbox
  5. Use of Pseudo-elements

Code Refactoring

My first independent assaignment was to observe pre-written HTML and CSS code. While the website was running mostly functional, I was able to refactor its original code into a more professional and organized manner. This included consolidating elements with identical properties, adding alt tags to all images and icons, and replacing div tags with appropriate markups.
